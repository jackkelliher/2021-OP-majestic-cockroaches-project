# Run

### Run the server

-   php artisan serve

### Run the test

-   php artisan dusk
-   or
-   php artisan dusk tests/Browser/xxxxxxx.php

### Update CSS

-   edit the file resources\sass\styles.scss
-   npm run dev
-   this will convert the scss file to css and add it to the public folder
